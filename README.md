# Arxiv Daily Paper Agent

这是一个自动化的 Python AI Agent 脚本。它的主要目标是：
1. 每天从 arXiv 获取 `cs.AI` 方向的最新论文。
2. 利用大语言模型（LLM）进行分析和过滤，剔除低质量（水文）论文。
3. 从高质量论文中提取有价值的技术思路和技巧。
4. 生成 Markdown 格式的日报，并通过 Webhook 自动推送到飞书群。

## 快速开始

### 1. 环境准备
```bash
# 复制配置文件模板
cp .env.template .env
```
编辑 `.env` 文件，填入你的大模型 Base URL、API Key 以及飞书群的 Webhook URL。

### 2. 安装依赖
本项目使用 `uv` 进行依赖管理，并已配置清华源：
```bash
uv sync
```

### 3. 本地启动与 API 触发

本项目已经集成了 **FastAPI**，您可以启动本地 Web 服务，并通过 API 接口手动触发论文的抓取和总结任务。

启动服务：
```bash
uv run main.py
```
服务默认运行在 `http://127.0.0.1:8000`。
启动时，程序会自动在后台启动一个 **APScheduler**，默认每天早上 09:00 (Asia/Shanghai) 自动触发昨日新论文的抓取、过滤与报告推送。
> **💡 断点与宕机恢复机制**：应用每次启动时会自动检查当前时间：如果发现已经跨过早上 9 点，并且昨天的数据还没生成报告，程序会**立刻启动一次自动补捞**，确保您不会因为服务重启而漏掉当天的日报推送。

#### 手动触发每日任务 (API)
接口提供了一个后台任务端点，您可以通过 HTTP POST 请求立即执行。默认抓取前一天的论文。

```bash
# 触发默认任务（前一天）
curl -X POST "http://127.0.0.1:8000/api/run"

# 通过指定参数触发某一天
curl -X POST "http://127.0.0.1:8000/api/run?target_date=2023-10-01"
```

您也可以在浏览器中访问 [http://127.0.0.1:8000/docs](http://127.0.0.1:8000/docs) 以查看可视化的 Swagger 接口文档，并直接在网页上发起测试请求。
